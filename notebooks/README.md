# Notebooks

Crear durante el proyecto:

| Notebook | Responsable | Día |
|----------|-------------|-----|
| `01_synthetic_validation.ipynb` | Persona A | Día 3 |
| `02_lalonde_eda.ipynb` | Persona C | Día 1 |
| `03_final_results.ipynb` | Persona C (consolidador) | Día 5-6 |

Convención: numerar para orden de lectura, código limpio (variables descriptivas, comentarios mínimos pero claros), output siempre al final.
